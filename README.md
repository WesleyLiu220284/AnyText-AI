# 🧠 AnyText-AI: 系统级无感外脑

> 拒绝切屏焦虑。一个藏在 Windows 底层的零 UI 知识引擎。

![AnyText-AI Banner](这里后续放你设计的Banner图片链接)

## ✨ 核心理念
打断心流，是工作中最反人类的设计。
**AnyText-AI** 不是一个传统的软件，它没有繁杂的界面，纯静默潜伏在系统后台。无论是冗长的学术文献、复杂的英文财报，还是生涩的代码报错，只要能复制，它就能在光标处瞬间为你吐出精准解答。

## 🚀 核心特性 (Features)
* **零 UI 干扰：** 纯后台运行，不占用桌面空间，不打断当前工作流。
* **万物皆可解：** 不挑软件环境，网页、PDF、微信对话框全场景通杀。
* **极速响应：** 选中 -> `Ctrl+C` -> `Ctrl+V`，3步魔法，直接在输入框获得结构化答案。
* **超轻量级：** 仅几十 KB 的脚本文件，无需安装臃肿的环境。

## 🛠️ 如何部署 (Quick Start)

### 1. 准备 API 密钥
本项目底层调用大模型接口。请前往 [智谱AI开放平台](https://open.bigmodel.cn/) 注册并获取你的专属 API Key。

### 2. 配置主程序
1. 下载本仓库中的 `.bat` 文件。
2. 右键选择“编辑”（用记事本打开）。
3. 找到 `$apiKey = "YOUR_API_KEY_HERE"` 这一行，将 `YOUR_API_KEY_HERE` 替换为你刚才申请的密钥。
4. 保存并关闭文件。

### 3. 运行体验
双击运行该 `.bat` 文件。听到系统提示音“滴”声后，说明隐形外脑已就位！
找一段复杂的文字，复制它，然后找个空白处粘贴，见证魔法。
